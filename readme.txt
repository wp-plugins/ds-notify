== Description ==
Ds notify - Frontend internal and web notifications!

DS Notify is a simply and intuitive plugin to display notifications on your frontend website

        Settings & features:

1.      Colour settings
2.      4 different internal notifications
3.      Cookie Settings
4.      Duration and delay setting
5.      Web notification set icon
6.      Add notification sound link
7.      Enable / disable internal notifications
8.      Enable / disable web notifications
9.      Add Custom notifications with multiple posts ,pages and type of notification selection
10.  	Custom position for your internal notification
11. 	AdBlock Safe

The plugin does not require any shortcode to make it work And it does not require any php/technical knowlegde to make it work.

== Installation ==

1. Open your Wordpress administration DASHBOARD 
2. go to Plugins then Add new and then click on Upload Plugin 
3. Choose the zip file which you downloaded from Codecanyon (ds-notify.zip)
4. When upload finishes click on Activate Plugin
5. Go to Notify Settings and fill the fields and choose the colors according to your choice.
6. Default Settings:
	A. Notify icon: https://cdn2.iconfinder.com/data/icons/facebook-svg-icons-1/64/notifications-512.png
	B. Notify Delay time: 3000
	C. Notify Duration time: 5000
	D. Notify sound url: http://oringz.com/oringz-uploads/sounds-937-job-done.mp3
	E. Enable web notifications?: Checked
	F. Enable internal notifications?: Checked
	G. Information notification color: #3596ba
	H. Warning notification color: #d3721d
	I. Error notification color: #c62317
	J. Success notification color: #32a34a
	K. Internal notificaiton posititon: Top left
	L. Enable internal notificaitons close button? : Checked
	M. Enable Notifications cookies? : Checked.
